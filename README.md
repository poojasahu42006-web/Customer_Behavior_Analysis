## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data and ending with interactive business insights. The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation in Power BI, report preparation, and a presentation created using Gamma.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.
